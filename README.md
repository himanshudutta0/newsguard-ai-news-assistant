# 🛡️ NewsGuard – Your AI-Powered News Assistant

**NewsGuard** is an AI-powered news analysis system that collects, processes, analyzes, and summarizes news articles in real-time. It empowers readers, researchers, and media analysts with deep insights on bias, sentiment, topics, and credibility in news content.

## 🚀 Features

- 🔍 **News Scraper**: Automatically fetches articles from multiple RSS feeds  
- ✂️ **Text Preprocessing**: Cleans and prepares content for analysis  
- 🧠 **Named Entity Recognition (NER)**: Identifies people, places, and organizations  
- 🗂️ **Topic Classification**: Categorizes news into relevant domains (e.g., Politics, Sports)  
- 🧵 **Topic Modeling & Timeline**: Extracts hidden topics using LDA and visualizes trends  
- 📰 **Summarization**: Generates concise summaries of lengthy articles  
- 🎭 **Sentiment Analysis**: Detects tone using VADER and TextBlob  
- ⚖️ **Bias Detection**: Flags potential political or opinion-based bias  
- 📊 **Streamlit Dashboard**: Interactive UI for exploring processed news articles  

## 🧪 Tech Stack

- **Python** – Core backend  
- **Streamlit** – Interactive dashboard  
- **SpaCy, Transformers** – NLP tasks  
- **TextBlob, VADER** – Sentiment analysis  
- **newspaper3k, feedparser** – News scraping  
- **Plotly, Matplotlib** – Data visualization  

## 📘 Use Cases

- 🧾 **Media Analysis**: Track bias, sentiment, and credibility across news sources  
- 🧠 **Academic Research**: Study political narratives, sentiment shifts, and topic trends  
- 📚 **Quick News Digests**: Generate concise summaries for fast reading  
- 🧰 **Tool for Journalists**: Identify key entities and topics in real-time  
- 📊 **Decision Support Dashboard**: Visualize sentiment and trends for strategic insights
